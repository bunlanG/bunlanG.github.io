Le design du site provient du site HTML5 UP, plus précisément du modèle Prologue, que j'ai modifié selon mes besoins.

Les icônes se trouvant dans le menu, ainsi que dans le CV, proviennent du projet Font-Awesome, qui utilise une fonte et du CSS pour générer ces icônes.

Ce site est généré avec Jekyll, un générateur de site Web statique.

Ce sité est hébergé sur GitHub, qui permet de créer un site personnel lié à son compte.
